Azure Learning Roadmap

For Azure DevOps Senior Platform Engineers

Learning Path Overview

Embark on a comprehensive journey through Microsoft Azure, tailored specifically for Senior Platform Engineers in the retail sector. This program equips you with the skills and knowledge needed to leverage Azure's powerful cloud capabilities. Each session is 30-60 minutes long, designed to run outside of office hours, ensuring a balance between learning and professional commitments.

Week 1: Core Azure Tools & Cloud Fundamentals

Azure Portal (Day 1)
  • Master the Azure user interface, manage resources efficiently, and explore key services such as Virtual Machines, App Services, and Azure Storage.
  • Learn to navigate the Azure portal effectively and understand resource management principles.
GitHub & GitHub Actions (Day 2)
  • Dive into version control with Git, create and manage repositories, and automate workflows using GitHub Actions.
  • Set up CI/CD pipelines for streamlined deployments to Azure.
  • Learn best practices for collaborative development and code review processes.
SSMS & Azure SQL Databases (Day 3)
  • Explore SQL Server Management Studio, connect to Azure SQL Databases, and perform essential cloud-based database management tasks.
  • Understand data migration strategies and security best practices for Azure SQL.

Hands-on Project: Deploy a basic web application to Azure App Service using GitHub Actions for continuous deployment.

Week 2: DevOps & Automation

Azure DevOps (Day 1)
  • Explore Azure DevOps services including Pipelines, Repos, Artifacts, and Boards.
  • Learn to create comprehensive CI/CD pipelines and manage projects using agile methodologies.
  • Understand how to integrate Azure DevOps with other Azure services for seamless workflow automation.
Kubernetes on Azure (AKS) (Day 2)
  • Dive into containerization with Docker and orchestration with Kubernetes.
  • Set up and manage Azure Kubernetes Service (AKS) clusters, deploy applications, and implement scaling and high availability strategies.
  • Learn about microservices architecture and how to effectively manage containerized applications in Azure.
GitHub Actions (Advanced) (Day 3)
  • Build on your GitHub Actions knowledge with advanced workflow automation, including multi-job pipelines, environment management, and integration with external services.
  • Learn to create custom GitHub Actions and optimize workflow performance.

Hands-on Project: Containerize an application, deploy it to AKS, and set up a full CI/CD pipeline using Azure DevOps and GitHub Actions.
